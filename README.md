# Engineering_Design_Project_Simulator
Made an simulator software that can do calculations and simulations for me instead of doing it manually via excel as done in university. 

An interactive engineering tool for analyzing projectile motion, static beam loading, dynamic impact loading (DMF), and material failure predictions, inspired by engineering design project coursework.
Built using Python, NumPy, Matplotlib, and a fully interactive Gradio GUI.

---

# Features:
## 1. Projectile Simulator<br>
  -No-drag + drag trajectories <br>
  -Comparison plot <br>
  -Adjustable: velocity, angle, mass, Cd, air density, diameter <br>
  -Shows both drag & no-drag ranges <br>

## 2. Multi-Angle / Multi-Velocity Optimizer<br>
  -Sweep angles or velocities <br>
  -Plots all trajectories <br>
  -Automatically highlights the maximum-range trajectory <br>

## 3.Static Load Failure Analysis
  -Computes static bending stress and failure load <br>
  -Supports rectangular & circular cross-sections <br>
  -MM-based input (mm, MPa) to match lab data <br>
  Includes: <br>
  -Load vs Span graph <br>
  -Load vs Square Section Size graph <br>

## 4. Dynamic Impact (DMF) Failure Analysis<br>
  -Impact velocity from drop height<br>
  -Static deflection<br>
  -DMF (Dynamic Magnification Factor)<br>
  -Dynamic stress under sudden loading<br>
